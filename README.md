## Project Overview
This project analyzes pizza sales data using SQL to uncover business insights such as revenue trends, top-selling pizzas, customer preferences, and sales performance.

## Objectives
- Analyze overall sales performance
- Identify top-selling pizza categories and sizes
- Calculate revenue and order trends
- Extract meaningful business insights using SQL queries

## Tools & Technologies
- SQL
-  MySQL Workbench

## Dataset
The dataset contains information about:
- Orders
- Order details
- Pizza types
- Pizza sizes
- Prices
- Categories

## Key Analysis Performed
- Total Revenue Analysis
- Top 5 Best-Selling Pizzas
- Revenue by Pizza Category
- Orders by Time and Date
- Average Order Value
- Most Popular Pizza Sizes
- Daily and Monthly Sales Trends

## SQL Concepts Used
- SELECT Statements
- WHERE Clause
- GROUP BY
- ORDER BY
- Aggregate Functions
- JOINS
- Subqueries


## Learning Outcome
This project improved my understanding of:
- Writing optimized SQL queries
- Data analysis using SQL
- Business-oriented data insights
- Database querying techniques

## Inspiration
This project was created for learning and portfolio purposes using publicly available datasets.
